# advanced-multivariate-analysis-stat-analysis-template
Template repository provided for the final Statistical Analysis and Report. 
## Data
Raw data must be downloaded manually from GESIS before running the scripts:
- EB67.1 (2007): https://search.gesis.org/research_data/ZA4529
- EB79.2 (2013): https://search.gesis.org/research_data/ZA5688
Place both .sav files in data/raw/
